Key Decisions

Engineering tradeoffs, explained

Performance

10x Faster, 30x Cheaper

Migrated all pipeline nodes from gemini-3-flash-preview to gemini-2.5-flash-lite. End-to-end latency dropped from 67s to under 7s. Cost per entry dropped from $0.009 to $0.0003.

6.66spipeline latency (from 67s)
Architecture

Acknowledge-Fast, Process-Slow

Railway's proxy breaks SSE connections. Switched to instant-acknowledge with background pipeline processing. Frontend polls status with silent sync — no disruptive reloads.

ML/Prompt Eng

Eval-Driven Entity Extraction

41-test harness across 10 failure families. Iterated from 75.6% → 100% pass rate (F1: 0.90 → 1.0). Targeted invalid-output examples beat abstract rules for steering LLM behavior.

100%pass rate (from 75.6%)
ML/Retrieval

3-Stage Entity Linking

Base-normalized exact → project-normalized → semantic embedding with acceptance gating. Handles "MindGraph" vs "mindgraph" vs "Mind Graph" without false merges on "Mind" vs "MindGraph".

Architecture

Temporal Query Routing

Time-based questions ("this week", "last 7 days") bypass the entire embedding/vector/rerank pipeline. Direct date-range DB query using entry summaries — faster and more complete.

Infrastructure

Ask Memory Compaction

Conversation memory with long-term compaction. Format compliance improved from 36.1% to 91.7%. Hallucination score maintained at 1.000 (perfect) across 50-case eval.

91.7%format compliance (from 36.1%)
